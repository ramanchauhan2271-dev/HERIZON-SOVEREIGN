## 🔮 What is HERizon Sovereign?

Women perform billions of hours of unrecognized work — mentoring, organizing, emotional support, crisis management — that never appears on a resume or LinkedIn profile.

**HERizon Sovereign** is a blockchain-powered Personal Reputation Vault that:
- 📝 **Captures** invisible labor using AI text analysis
- 🏅 **Converts** it into permanent Soulbound Tokens (SBTs) on the blockchain
- 💰 **Quantifies** its monetary value using professional rate proxies
- 📊 **Advocates** for you with AI-generated promotion reports
- 🤝 **Verifies** contributions through NFC-based peer attestations

Your reputation. Your proof. Fully sovereign.

## ✨ Key Features

| Feature | Description |
|---|---|
| 🔐 **Self-Sovereign Identity** | Login without Big Tech — your identity, your control |

| ⛓️ **Soulbound Tokens (ERC-5114)** | Non-transferable blockchain credentials on Polygon |
| 📡 **NFC Tap to Vouch** | Offline peer verification via physical tap |
| 🤖 **LangChain AI Advocate** | Auto-generates promotion & impact reports |
| 🌸 **Impact Garden** | Visual dashboard — each flower = one verified contribution |
| 💎 **Relational Equity Score** | Quantified measure of your professional impact |
